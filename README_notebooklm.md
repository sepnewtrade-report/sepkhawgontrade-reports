# คู่มือการใช้งานเครื่องมือลบ Notebook ใน NotebookLM อัตโนมัติ (Auto-Delete)

เครื่องมือนี้ใช้สำหรับดึงรายการและลบ Notebook ใน Google NotebookLM ตามเงื่อนไขที่คุณเลือก (ลบทั้งหมด, ลบตามคำค้นหาในชื่อ, ลบตามอายุวัน หรือเลือกลบเป็นรายตัว) ผ่านโปรแกรมภาษา Python ด้วยไลบรารี `notebooklm-py`

---

## 🛠️ ขั้นตอนการติดตั้งและตั้งค่า (Setup)

เนื่องจากสคริปต์นี้เขียนด้วย Python จึงแนะนำให้รันภายใต้ **Virtual Environment (.venv)** เพื่อป้องกันไม่ให้กระทบกับส่วนอื่น ๆ ของระบบ

### 1. สร้างและเปิดใช้งาน Virtual Environment
เปิด Terminal ขึ้นมาแล้วรันคำสั่งดังต่อไปนี้ตามลำดับ:

```bash
# 1. สร้างโฟลเดอร์ Virtual Environment ชื่อ .venv
python3 -m venv .venv

# 2. เปิดใช้งาน (Activate) Virtual Environment
source .venv/bin/activate
```
*(เมื่อเปิดสำเร็จ คุณจะเห็นตัวอักษร `(.venv)` แสดงอยู่ที่ด้านหน้าบรรทัดของ Terminal)*

### 2. อัปเกรด Pip และติดตั้งไลบรารีที่จำเป็น
เมื่ออยู่ใน venv แล้ว ให้รันคำสั่งเหล่านี้เพื่อติดตั้ง:

```bash
# อัปเกรด pip ให้เป็นเวอร์ชันล่าสุด
pip install --upgrade pip

# ติดตั้งไลบรารี notebooklm-py
pip install notebooklm-py
```

## 🔑 ขั้นตอนการเข้าสู่ระบบ (Authentication)

เนื่องจาก NotebookLM ไม่มี API อย่างเป็นทางการสำหรับบัญชีทั่วไป ไลบรารีนี้จึงทำการจำลองเซสชันด้วยคุกกี้เบราว์เซอร์ ซึ่งคุณต้องทำเข้าสู่ระบบครั้งแรกก่อนหนึ่งครั้งดังนี้:

1. รันคำสั่งเข้าสู่ระบบใน Terminal:
   ```bash
   notebooklm login
   ```
2. หน้าต่างเบราว์เซอร์จะเปิดขึ้นมาอัตโนมัติ ให้คุณเข้าสู่ระบบบัญชี Google ที่เชื่อมโยงกับ NotebookLM ของคุณตามปกติ
3. เมื่อเข้าสู่ระบบสำเร็จแล้ว ให้ปิดหน้าต่างเบราว์เซอร์นั้นได้เลย ระบบจะบันทึกคุกกี้เซสชันไว้ในเครื่องของคุณโดยอัตโนมัติ (ซึ่งพร้อมใช้งานสำหรับสคริปต์ทันที)

---

## 🚀 วิธีรันสคริปต์ลบ Notebook

เมื่อเข้าสู่ระบบเรียบร้อยแล้ว คุณสามารถรันสคริปต์ลบได้เลยด้วยคำสั่ง:

```bash
python delete_notebooks.py
```

### เงื่อนไขที่มีให้เลือก:
1. **Delete ALL notebooks (ลบทั้งหมด)**: ลบทุก Notebook ที่มีอยู่ในบัญชีของคุณทันที
2. **Delete by keyword in Title (ลบตามคำเฉพาะเจาะจง)**: ระบุคำค้นหา (เช่น "daily") สคริปต์จะค้นหาและแสดงรายการ Notebook ที่มีคำนั้นในชื่อ ก่อนจะให้ยืนยันการลบ
3. **Delete by age / older than N days (ลบที่เก่ากว่า N วัน)**: กำหนดจำนวนวัน (เช่น 7 วัน หรือ 30 วัน) เพื่อลบ Notebook เก่า ๆ ที่ไม่ได้ใช้งานแล้ว
4. **Interactive selection (เลือกทีละรายการ)**: ระบบจะถามคุณทีละตัวว่าต้องการลบ Notebook นี้หรือไม่ (`y` = ลบ, `n` = ข้าม, `q` = ยกเลิก)

### ⚠️ ระบบความปลอดภัยในการยืนยัน (Double Confirmation)
เพื่อป้องกันไม่ให้คุณเผลอลบข้อมูลสำคัญโดยไม่ตั้งใจ เมื่อเลือกเงื่อนไขการลบเสร็จแล้ว ระบบจะบังคับให้พิมพ์คำว่า `DELETE` ตัวพิมพ์ใหญ่ทั้งหมดเพื่อยืนยันขั้นสุดท้ายในการเริ่มลบข้อมูล
