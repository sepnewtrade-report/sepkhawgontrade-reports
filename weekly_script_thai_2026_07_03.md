## 📌 บทนำ (Hook & Intro)
*(เวลาแนะนำ: 0:00 - 1:15)*

**[ผู้ดำเนินรายการมองกล้องด้วยสีหน้ายิ้มแย้ม กระฉับกระเฉง พลังเสียงเต็มอิ่ม ชวนติดตาม / ขึ้นกราฟิกดัชนี SET Index ปิดที่ระดับ 1,611.28 จุด และขึ้นตัวหนังสือเด่นกลางจอ "ต่างชาติซื้อกระหน่ำ ดันบาทอ่อนหนุนส่งออก หุ้นใหญ่บวกแรง!" พร้อมภาพประกอบตารางค่าเงินบาทที่ปิดที่ 33.38 บาทต่อดอลลาร์สหรัฐ และกลุ่มหุ้นนำตลาดอย่าง IVL, KBANK, CPALL, GULF]**

"สวัสดีครับนักลงทุนทุกท่าน ยินดีต้อนรับเข้าสู่รายการ **เสพข่าวก่อนเทรด หุ้นไทย** สรุปวิเคราะห์เจาะลึกภาพรวมตลาดหุ้นไทยประจำสัปดาห์ที่จะช่วยให้คุณก้าวทันทุกความเคลื่อนไหวของเม็ดเงินลงทุนแบบมืออาชีพครับ!

ในรอบสัปดาห์ที่ผ่านมา (29 มิถุนายน - 3 กรกฎาคม 2569) ตลาดหุ้นไทยเคลื่อนไหวในทิศทางที่แข็งแกร่งอย่างยิ่งครับ ดัชนี SET Index ปรับตัวขึ้นทะยานยืนเหนือระดับ 1,600 จุดได้อย่างงดงาม โดยปิดสัปดาห์ที่ 1,611.28 จุด หรือเพิ่มขึ้นสะสมรายสัปดาห์ถึง +4.47% คิดเป็นเพิ่มขึ้นกว่า +68.94 จุดครับ!

ความเคลื่อนไหวสำคัญของโลกสัปดาห์นี้เกิดขึ้นก่อนช่วงหยุดยาววันชาติสหรัฐฯ ครับ เมื่อตัวเลขการจ้างงานนอกภาคเกษตร หรือ **Non-farm Payrolls** เดือนมิถุนายนของสหรัฐฯ ประกาศออกมาต่ำกว่าคาดการณ์อย่างรุนแรง เพิ่มขึ้นเพียง 57,000 ตำแหน่งเท่านั้น! ตัวเลขที่ย่ำแย่นี้สร้างแรงสั่นสะเทือนทำให้นักลงทุนเก็งว่าเฟดจะต้องปรับลดดอกเบี้ยนโยบายเร็วขึ้น (Dovish Shift) ส่งผลให้เกิดปรากฏการณ์ **Sector Rotation** หมุนเวียนเม็ดเงินลงทุนออกจากกลุ่มเทคโนโลยีที่มีระดับราคาตึงตัว เข้าสู่กลุ่มอุตสาหกรรมดั้งเดิม หุ้นคุณค่า และตลาดเกิดใหม่เอเชีย รวมถึงตลาดหุ้นไทยด้วยครับ

สำหรับตลาดหุ้นไทย สภาวะดังกล่าวส่งผลให้มีเม็ดเงินลงทุนต่างชาติไหลเข้าอย่างท่วมท้น โดยในรอบสัปดาห์นี้นักลงทุนต่างชาติซื้อสุทธิสะสมในตลาดหุ้นไทยสูงถึงประมาณ 28,000 ล้านบาท! และค่าเงินบาทที่แกว่งตัวอ่อนค่าในกรอบ **33.30 - 33.45 บาทต่อดอลลาร์** ตลอดสัปดาห์ ก่อนจะปิดที่ 33.38 บาทต่อดอลลาร์ ก็ส่งผลบวกในรูปแบบกำไรจากอัตราแลกเปลี่ยน (FX Gain) ให้กับกลุ่มส่งออกอาหารและเครื่องดื่มอย่าง **SAPPE** ที่ปรับตัวบวกเด่น ขณะเดียวกัน ความต้องการซื้อสะสมหุ้นใหญ่และกลุ่มปลอดภัยช่วยหนุนราคาหุ้นอย่าง **IVL**, **KBANK**, **CPALL**, **TOP** รวมถึง **GULF** ที่เสร็จสิ้นการควบรวมกิจการกับ INTUCH ให้ดีดตัวนำตลาดอย่างร้อนแรงครับ

สัปดาห์หน้าโอกาสในการลงทุนของหุ้นไทยจะอยู่ตรงไหน? ทิศทาง Fund Flow ต่างชาติจะดันดัชนีไปได้ไกลเพียงใด และเราต้องเตรียมพร้อมแผนการเทรดอย่างไร? มาเจาะลึกไปพร้อมๆ กันเลยครับ!"

## 1️⃣ ภาพรวมตลาดการเงินไทยประจำสัปดาห์ (Weekly Market Overview)
*(เวลาแนะนำ: 1:15 - 3:15)*

**[ขึ้นกราฟิกสถิติดัชนีหลักรายสัปดาห์: SET Index / mai / ยอด Fund Flow ต่างชาติ / อัตราแลกเปลี่ยนเงินบาท / สินค้าโภคภัณฑ์ พลังงานและทองคำ]**

"ภาพรวมตลาดการเงินและหุ้นไทยในสัปดาห์นี้คึกคักอย่างโดดเด่นจากเม็ดเงินลงทุนต่างชาติไหลเข้าหนาแน่นและการปรับพอร์ตของนักลงทุนสถาบันครับ:
*   **ดัชนี SET Index:** ปิดสัปดาห์ ณ วันศุกร์ที่ 3 กรกฎาคม 2569 ที่ระดับ **1,611.28 จุด** ปรับตัวเพิ่มขึ้นจากสิ้นสัปดาห์ก่อน **+68.94 จุด หรือ +4.47%** มูลค่าการซื้อขายในวันสุดท้ายหนาแน่นขึ้นแตะระดับ **80,492.88 ล้านบาท**
*   **ตลาด mai:** แกว่งตัวฟื้นตัวเป็นบวกตามบรรยากาศตลาดโดยรวม (Sentiment) ที่สดใสขึ้น
*   **Fund Flow ต่างชาติ:** นักลงทุนต่างชาติมีสถานะ **ซื้อสุทธิสะสมในตลาดหุ้นไทยประมาณ 28,000 ล้านบาท** ในรอบสัปดาห์ (โดยเฉพาะในวันศุกร์ที่ 3 กรกฎาคม 2569 มียอดซื้อสุทธิสูงถึง **5,782.12 ล้านบาท** ในตลาด SET) รับผลของการโยกย้ายเงินลงทุนเข้าสู่ตลาดเอเชีย
*   **ค่าเงินบาท:** เคลื่อนไหวอ่อนค่าอยู่ในกรอบ **33.30 - 33.45 บาทต่อดอลลาร์สหรัฐ** โดยปิดสัปดาห์ที่ระดับ **33.38 บาทต่อดอลลาร์** ซึ่งช่วยหนุนรายรับของผู้ส่งออกกลุ่มอาหารแปรรูป
*   **ราคาสินค้าโภคภัณฑ์โลก:** ราคาน้ำมันดิบ Brent ปรับตัวลดลงปิดที่ **$71.80 ต่อบาร์เรล** จากความกังวลอุปสงค์ชะลอตัวและการค้าน้ำมันเบนซินของรัสเซียกับอินเดีย ส่วนราคาทองคำ Spot ทะยานขึ้นมาปิดแกร่งแถว **$4,045.00 ต่อออนซ์** รับจิตวิทยาผ่อนคลายของดุลนโยบายการเงินสหรัฐฯ และ Bitcoin ปิดยืนหยัดที่ **$61,350** ครับ"

4.  **กลุ่มชิ้นส่วนอิเล็กทรอนิกส์โลกผันผวนแต่ดีดตัวกลับช่วงปลายสัปดาห์** 🟡

7.  **ความต้องการสินค้าส่งออกกลุ่มอาหารแปรรูปและเครื่องดื่มยังคงขยายตัวเด่น** 🟢
    *   ยอดคำสั่งซื้อล่วงหน้าของสินค้าเครื่องดื่มชูกำลัง น้ำมะพร้าว และปลาทูน่าแปรรูปไปยังสหรัฐฯ และยุโรปยังคงเติบโตอย่างมั่นคง
8.  **จำนวนนักท่องเที่ยวต่างชาติสะสมในพื้นที่ท่องเที่ยวหลักยังคงยืนแข็งแกร่ง** 🟢
    *   มาตรการอำนวยความสะดวกด้านวีซ่าและจำนวนเที่ยวบินที่ฟื้นตัว หนุนยอดจำนวนนักท่องเที่ยวต่างชาติเข้าประเทศอย่างสม่ำเสมอ ส่งผลดีต่อกลุ่มโรงพยาบาลพรีเมียมและกลุ่มโรงแรม
9.  **สถาบันและต่างชาติเข้าซื้อสะสมหนาแน่นในช่วงรอยต่อสิ้นไตรมาส 2** 🟢
    *   สถาบันในประเทศประสานพลังกับกองทุนต่างชาติเข้าซื้อสะสมหุ้นใหญ่เพื่อปรับน้ำหนักการลงทุน รับทิศทางเม็ดเงินลงทุนใหม่ที่จะไหลเข้าสู่เอเชีย
10. **การปรับตัวตามรอบอิทธิพล Capricorn Full Moon และดาวพุธถอยหลัง** 🔴/🟡
